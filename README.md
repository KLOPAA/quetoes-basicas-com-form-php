No script PHP (atv3.php), deve-se avaliar o valor recebido: se a idade for igual ou superior a 18 anos, exiba a frase "Você é maior de idade" na cor verde; se for inferior, exiba "Você ainda é menor de idade" na cor vermelha. 
Utilize estruturas condicionais (if/else) e CSS inline para a estilização.
